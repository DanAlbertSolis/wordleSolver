WORDLE SOLVER

This chrome extension is meant to assist wordle puzzles by giving possible words based on user input. 

Correctness of each letter is handled by the colors black green and yellow. The arrays are reset each time the chrome extension is opened. 

sgb-words contains all 5 letter words in the English language. To change the language simply replace the file with the equivalent. 

The sort fodler includes a python script to pull all 5 letter words from a language and store it in a separate txt file.
